# Raw Data

Raw ecommerce behavior files are intentionally excluded from Git.

Use the downloader:

```bash
python src/download_sample.py --rows 200000
```

Primary source: REES46 / Kaggle multi-category ecommerce behavior dataset.
Practical mirror used by this project:
`https://huggingface.co/datasets/kevykibbz/ecommerce-behavior-data-from-multi-category-store_oct-nov_2019`
